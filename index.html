HTML has allowed widespread access to the internet on an international scale. In addition to access, this provided the opportunity for digital technologies to become a prominent tool in mass communication. In the beginning, HTML allowed earlier models of computers to be able to access information such as files and email. This opened a gateway for computers to be able to communicate with one another. In the 1980s, computers communicated using XML to display information on webpages. The first verison of HTML (HTML 1.2) was written in 1993 by Tim Berners-Lee.By the early 2000s, most web pages were marked up using HTML 4.01. These earlier versions of HTML had limitations on what information could be displayed.

For example, HTML 1.2 allowed for basic headings, paragraphcs, anchors/links, lists, and images. HTML4.01 introduced the ability to stylize words on pages by bolding, italicizing, and underlining them as an examples. You could also include styling by embedding Cascading Style Sheets (CSS) and JavaScript (JS). Utilizing these additional features allowed pages to look more structured and become more interactive for users who accessed pages. As the internet and the way we communicate developed over time, HTML5 was then updated to allow us the option to include aduio, video, and canvas in addition to types of media we are able to display now.
